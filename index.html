General information about the *.chm format

HTMLHelp (Microsoft Compressed HTML Help, Microsoft Compiled HTML Help,. CHM) - a proprietary file format for context-sensitive help, developed by Microsoft and released in 1997 as a replacement format WinHelp. Contains a set of HTML-pages can also include links to content pages, index, as well as a base for full-text search on content pages. All members of the. CHM files are compressed by the algorithm LZX.

CHM. Advantages and disadvantages

The advantages of the format: File size is less than the normal HTML We use all the formatting options available in HTML and CSS Full-text search Ability to view the set. Chm-files as one, with the general contents and an index (such as early versions of MSDN Library are provided in a format HTMLHelp)

Disadvantages of format: Undocumented. It is well known to fail, which does not view these files. This is easily remedied by running the command «regsvr32% windir% \ system32 \ hhctrl.ocx». Files can not be opened, if the path to the file has Cyrillic characters, and symbols "_" and "#". The format is focused only on Windows.
